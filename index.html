Endpoints Overview

MethodEndpointDescription
POST/api/registerRegister a new user
POST/api/loginLogin and receive JWT token
POST/api/files/uploadUpload a file
GET/api/filesGet all uploaded files
GET/api/files/:idFetch file details
DELETE/api/files/:idSoft delete file
PATCH/api/files/restore/:idRestore soft-deleted file
GET/api/admin/usersFetch all users (admin only)
PATCH/api/admin/users/restore/:emailRestore soft-deleted user
